DEFRAG 개인정보 처리방침

시행일: 2026-05-08

1. 수집하는 개인정보

  • Apple Sign-In 식별자(uid)
  • 회원이 입력한 습관·체크 기록·메모(서비스 제공 목적)
  • 로컬 알림 발송을 위한 시간 설정

이름·이메일은 Apple 의 비공개 릴레이를 사용해 직접 수집하지 않으며, Firebase Auth 가 발행한 익명 식별자(uid)만 보관합니다.

2. 이용 목적

회원의 습관 트래킹 데이터 동기화·백업과 로컬 알림 스케줄링에 사용합니다.

서비스 개선을 위한 익명 사용 통계(Firebase Analytics)는 제어판 → 익명 사용 통계 공유 토글이 ON 일 때만 작동하며, 기본값은 OFF 입니다. 토글을 끄면 즉시 전송이 중단됩니다.

3. 보관 기간

회원이 회원탈퇴를 요청하는 즉시 모든 데이터(Firestore /users/{uid} 이하 전체, Firebase Auth 사용자 레코드)를 삭제합니다.

4. 외부 처리

데이터 저장은 Google Firebase(미국·아시아 리전)를 위탁합니다. 약관·SLA 는 Google 의 Firebase 데이터 처리 약관을 준용합니다.

5. 이용자의 권리

회원은 앱 내 "회원탈퇴" 메뉴로 본인 정보 열람·정정·삭제·처리정지를 요구할 수 있습니다.

6. 변경 고지

본 방침이 변경되면 시행일 7일 전까지 앱 내 또는 본 페이지에 공지합니다.
